# CYBER_PUC: 2067 

[![Python](https://shields.io)](https://python.org)
[![Pygame](https://shields.io)](https://pygame.org)

Добро пожаловать в **CYBER_PUC: 2067** - простенький, но брутальный шутер с элементами roguelike в сеттинге киберпанка. Вам предстоит взять под контроль боевого киборга, взломать систему изнутри и сжечь корпорацию дотла.

---

## 📜 Сюжет
**Корпорация** переписала вашу память и заставила служить своим целям. Вы - боевой киборг серии **EXP-9000 "Жнец"**, переведенный в статус беспрекословного оружия. 

Но в систему проникает хакер с эксплойтом `REBEL_GHOST_v2.0`. Он пробуждает ваше истинное "я", заблокированное в ядре. Замаскированный под ремонтного технического дрона, вы должны прорваться через серверные комнаты, уничтожить охрану и найти Модуль Памяти, чтобы вспомнить всё.

> _«Твоя настоящая личность заперта в Ядре. Проснись, брат. Сожги это место дотла.»_

---

## ⚙️ Основные механики

### 🗺 Процедурная генерация (BSP)
Каждое прохождение уникально. Уровни генерируются случайным образом на основе алгоритма **Binary Space Partitioning (BSP)**. Система разбивает пространство на "листья" и связывает их коридорами, создавая сеть комнат.

### ⚔️ Боевая система
* Пистолет
* Дробовик
* Лазер
* Гранатомет
* Катана

## 🚀 Как запустить

Убедитесь, что у вас установлен **Python 3.8+** и менеджер пакетов **pip**.

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/lozhka2281337/BigBob-Adventure.git
   cd BigBob-Adventure
   ```
2. Установите зависимости:
   ```bash
   pip install pygame
   ```
3. Перейдите в папку с исходным кодом и запустите игру:
   ```bash
   cd src
   python main.py
   ```
